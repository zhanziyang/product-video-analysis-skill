# Product Video Analysis Skill Design

## Goal

Create a public, owner-maintained MIT-licensed Agent Skill repository named `product-video-analysis-skill`, portable across Agent Skills hosts and enhanced for Codex with FFmpeg/Python workflows.

## Core rule

Every full analysis preserves all modules from the accepted OnePay standard. Animation terminology, timing curves, sound, and product mapping are additions, not replacements.

## Deliverables

- Portable `SKILL.md`
- Seven deep reference guides plus evidence labeling
- Report, shot, and animation templates
- JSON report contract
- OnePay worked excerpt
- Frame extraction, audio analysis, and report validation scripts
- Codex wrappers and workflow notes
- Unit tests and CI
- README and MIT license

## Quality contract

A complete report includes the 25 required modules, multi-frame evidence for major motion, evidence labels, screenshot-backed timing estimates, sound mapping, product-strategy analysis, implementation methods, exercises, user-product transfer, and rendered-PDF inspection.

## Compatibility

The skill follows the Agent Skills folder format. Automation uses Python 3.10+ standard library and FFmpeg/FFprobe. The core instructions remain usable without automation.

## Publication

The intended repository is `zhanziyang/product-video-analysis-skill`, public, maintained only by the owner, under MIT.
