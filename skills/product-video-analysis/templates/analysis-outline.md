# Complete Product Video Analysis Outline

Use this exact order unless the source demands a clearly justified variation.

1. Title and source
2. Evidence labels and limitations
3. Video metadata
4. Executive summary
5. Product background and core value
6. Communication objective
7. Creative proposition
8. Planning logic
9. Narrative structure
10. Copy architecture
11. Music, sound effects, and voice-over
12. Tempo and motion-density map
13. Visual design system
14. Transition system
15. Detailed timecoded breakdown
16. Motion-to-product mapping summary
17. Copy-image-motion-sound coordination matrix
18. Implementation methods
19. Project organization
20. Simplified recreation
21. Staged training plan
22. Transferable methodology
23. Application to the user's product
24. Continuous-frame appendix
25. Sources and uncertainty log

## Detailed segment contract

Every important segment contains:

- Time range
- Narrative job
- Screenshot sequence
- Observed action
- Animation name(s)
- Property changes
- Estimated seconds and frames
- Easing / timing graph
- Motion principles
- Composition / typography / color / depth
- Copy and sound
- Product-feature mapping
- Implementation method
- Reusable lesson
