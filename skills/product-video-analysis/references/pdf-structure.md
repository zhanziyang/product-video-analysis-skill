# PDF Structure

A full illustrated report commonly lands between 45 and 80 pages for a 20–60 second video. Page count is not a quality target; evidence and readability are.

## Recommended sequence

### Front matter

1. Cover
2. Method and evidence labels
3. Video metadata
4. Executive summary
5. Product truth and communication objective

### Strategy and macro structure

6. Creative proposition
7. Planning logic
8. Narrative act map
9. Copy ladder
10. Music/sound/VO map
11. Motion-density chart
12. Visual system: type, color, depth, brand objects
13. Transition grammar

### Detailed breakdown

For each segment:

- segment title and time range;
- 3–8 frame sequence;
- what happens;
- animation names;
- property changes;
- duration and frame estimate;
- timing function / graph description;
- motion principles;
- composition and typography;
- sound cue;
- copy-image-motion-sound coordination;
- product-feature mapping;
- implementation approach;
- reusable lesson.

### Production and learning

- technical pipeline;
- AE/3D/Figma/editing structure;
- simplified recreation;
- staged exercises;
- transferable method;
- application to the user's product;
- evaluation rubric.

### Appendix

- continuous frame contact sheets;
- transcript/copy list;
- sources and verification notes;
- uncertainty notes.

## Screenshot rules

- Preserve source aspect ratio.
- Add visible timecodes.
- Use before/middle/after frames for motion proof.
- Do not enlarge low-resolution crops beyond legibility.
- Avoid repeating the same frame merely to fill layout.
- Use callouts only when they identify a concrete change.

## Page QA

Render the PDF to page images and inspect:

- clipped text or images;
- overflowing tables;
- missing glyphs;
- tiny body copy;
- incorrect image crops;
- blank or duplicate pages;
- low-contrast labels;
- broken links;
- screenshots without timecodes;
- sections whose headings exist but whose content is absent.
